Installation Guide
================

Requirements
-----------

BeamZ requires Python 3.8 or higher and the following dependencies:

* NumPy >= 1.20.0

Basic Installation
----------------

You can install BeamZ using pip:

.. code-block:: bash

   pip install beamz

Development Installation
----------------------

To install BeamZ for development:

1. Clone the repository:
   .. code-block:: bash

      git clone https://github.com/QuentinWach/beamz.git
      cd beamz

2. Install the package in development mode with all development dependencies:
   .. code-block:: bash

      pip install -e ".[dev]"

This will install BeamZ along with all development tools needed for testing and documentation.
